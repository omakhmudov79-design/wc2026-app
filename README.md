# ⚽ ЧМ 2026 — AI Football Analysis App

World Cup 2026 AI-powered football analysis application built with Flutter and Claude API.

## 🚀 Quick Start

### Cloud Build (Recommended)
No need to install Flutter or Android Studio!

The app builds automatically in the cloud using GitHub Actions.

1. Push code to this repository
2. Go to **Actions** tab
3. Click **Run workflow**
4. Wait 15-20 minutes
5. Download APK from artifacts

### Local Build
```bash
flutter pub get
flutter pub run build_runner build
flutter run
```

## ✨ Features

- 🤖 AI team analysis with Claude
- 📊 Match predictions with probabilities
- 🔔 Push notifications
- 📅 Calendar integration
- 💾 Local storage
- 🌐 Offline mode
- 🎨 Material 3 design

## 📱 Requirements

- Flutter 3.16+
- Dart 3.0+
- Android SDK API 21+

## 🔑 Setup

1. Get API key from [api.anthropic.com](https://api.anthropic.com)
2. Open app → Settings → Enter API key
3. Select teams → Run Analysis
